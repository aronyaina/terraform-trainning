Fibonacci App

This app calculates the Fibonacci sequence up to a specified number.

To get the Fibonacci sequence up to a number, make a GET request to the following endpoint:

  • /fibonacci/:n
  • /fibonacci/

First route for calculation of the fibonnaci
Replace :n with the number you want to calculate the Fibonacci sequence up to.

Second route to get the fibonnaci already calculated , the number and the laste element of the fibonnaci.
